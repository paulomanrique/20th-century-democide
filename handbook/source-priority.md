# Source Priority

Use this order when recovering material:

1. Live original on `hawaii.edu/powerkills`.
2. Live original on self-authored blogs such as `freedomspeace.blogspot.com`.
3. Self-authored mirrors or republished copies that preserve the original text.
4. Wayback snapshots closest to the publication window or the latest intact capture.
5. Mark `blocked` and document the failure if the item cannot be recovered responsibly.
