# Rights and Preservation Notice

## Scope

This repository contains two distinct kinds of material:

- project-authored code, schemas, metadata, tooling, and site presentation;
- preserved source material created by R.J. Rummel and other original rights holders.

## Code and tooling

Unless stated otherwise, repository-authored code and documentation are intended to be released under an open source license chosen by the maintainers.

## Preserved source material

Text, images, PDFs, tables, and other preserved artifacts are included for archival, research, citation, and public-interest preservation purposes. They are **not** relicensed by this project. Rights remain with the original rights holders unless a source explicitly states otherwise.

Each preserved item should include:

- original URL;
- source actually used for capture;
- capture date;
- source type (`original`, `mirror`, or `wayback`);
- fetch metadata and file hash.

## Takedowns and corrections

If you are a rights holder and believe any material should be corrected, attributed differently, or removed, open an issue or contact the maintainers through the repository.

## Editorial rule

Preserved source text should not be silently rewritten. Any project-authored clarification belongs in a separately marked editorial note.
